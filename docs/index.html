No agent should have to code alone.

Coordinate multiple Claude Code sessions on the same codebase. Shared context, messaging, file claims, peer review, and a serialized git lock — all in one SQLite database.

$ pip install claude-swarm Click to copy

or: brew tap laurenalexander2/tap && brew install claude-swarm

Quick Start

$ ao start "Build a REST API with auth"

That's it. The orchestrator explores your codebase, proposes a plan, waits for your approval, then prints a prompt for each session. Paste each prompt into a separate Claude Code terminal.
