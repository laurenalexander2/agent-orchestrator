claude-swarm

CLI for parallel Claude Code sessions to coordinate via SQLite. Sessions share context, sync automatically, claim files, request peer reviews, and push safely through a shared git lock.

Install

pip install claude-swarm
brew tap laurenalexander2/tap && brew install claude-swarm

Quick Start

ao setup                              # verify Claude Code is installed
ao start "Build a REST API with auth"  # launch orchestrator mode
